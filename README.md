# Automation Testing Framework

## Project Goal
This project is an **Automation Testing Framework** that automatically tests a login page using **Python** and **Selenium**. It helps verify login functionality efficiently and reduces manual testing effort.

## Technologies Used
- Python 3.x
- Selenium WebDriver
- Chrome/Firefox Browser Driver

## Features
- Automated login tests for web applications
- Supports multiple browsers
- Easy to extend for additional test cases

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/automation-testing-framework.git

Navigate to the project folder:

cd automation-testing-framework

Install dependencies:

pip install -r requirements.txt

(Make sure Python and pip are installed)

Usage
Update the test script with your login page URL and credentials.

Run the tests:

python test_login.py
View the test results in the terminal.
